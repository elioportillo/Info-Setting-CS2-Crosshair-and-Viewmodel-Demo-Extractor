# Info Setting CS2

Extract CS2 crosshair codes and viewmodel settings directly from Counter-Strike 2 demo files.

## Live Website

https://infosettingcs2.online

<img width="1205" height="895" alt="image" src="https://github.com/user-attachments/assets/3cd4690f-b0fe-4b72-9847-f707f51520bc" />

## Features

* Upload any `.dem` Counter-Strike 2 demo file
* Extract player crosshair codes
* Extract player viewmodel settings
* Browse all players found in the demo
* Copy crosshair and viewmodel configurations with one click
* Direct access to player Steam profiles

## How It Works

1. Upload a Counter-Strike 2 demo file.
2. The demo is parsed using `demoparser2`.
3. Player data is extracted from the demo.
4. Crosshair and viewmodel settings are displayed instantly.
